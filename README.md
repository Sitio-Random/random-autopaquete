Hola! Para empezar necesitas tener instalado Node Js, el ID o editor de texto de preferencia ( Atom, Visual Studio Code, ect ) y Git como sistema de control de versiones. Una vez instalados, abre la terminal o línea de comandos y copia esto en el directorio donde quieres que se encuentre el proyecto para descarga el repositorio desde Github.
>git clone https://github.com/Autopaquete/random-autopaquete

Una vez descargado , en la terminal escribes 
>cd random-autopaquete
>npm install

Ya descargados las packages y dependencias necesarias escribes
>npm start

y se abrirá en el navegador localhost:3000 donde puedes empezar a trabajar. Puedes trabajar en cualquier rama ya hecha o crear una nueva.
